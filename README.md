# HarajApp
یک وب سایت ساخته شده با بک اند node.js و فریم ورک express.js با ویو ejs و پایگاه داده‌ی mongodb که هنوز ویوی آن کمی کار دارد و باید در رندر کردن بادی و هدر کمی تغییرات داد تا بهینه و ماژولار شود.
